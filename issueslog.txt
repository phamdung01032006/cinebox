Sửa như sau:

1. Trong [header.php](c:\xampp\htdocs\cinebox\includes\header.php), bỏ redirect cứng sang register  
Xóa đoạn này:

```php
if(!isset($_SESSION["userLoggedIn"])) {
    header("Location: register.php");
}
```

Thay bằng:

```php
$userLoggedIn = $_SESSION["userLoggedIn"] ?? null;
```

2. Chặn ở trang xem phim [watch.php](c:\xampp\htdocs\cinebox\watch.php)  
Thêm ngay đầu file (sau `require_once("includes/header.php");`):

```php
if(!isset($_SESSION["userLoggedIn"])) {
    header("Location: login.php?returnUrl=" . urlencode("watch.php?id=" . $_GET["id"]));
    exit();
}
```

3. (Khuyên dùng) Chặn luôn ở [entity.php](c:\xampp\htdocs\cinebox\entity.php) nếu bạn muốn bấm vào chi tiết cũng cần login  
Nếu không thì để public.
